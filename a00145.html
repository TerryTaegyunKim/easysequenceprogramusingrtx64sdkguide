Detailed Description

INI 파일을 파싱하여 설정 값을 가져오는 클래스입니다.

이 클래스는 INI 파일에서 특정 섹션 및 키에 대한 값을 읽어오는 기능을 제공합니다.

주어진 키 값이 기준 키 값과 지정된 모드에 따라 일치하는지 확인하는 함수입니다.

Parameters
key: 비교할 키 값입니다.
refKey: 비교 기준이 되는 키 값입니다.
mode: 비교 방법을 결정하는 모드입니다.
  • 0: 전체 키가 기준 키와 정확히 일치하는지 확인합니다.
  • 1: 구분자 앞부분이 기준 키와 일치하는지 확인합니다.
  • 2: 구분자 뒷부분이 기준 키와 일치하는지 확인합니다.
delimiter: 키와 기준 키를 구분하는 데 사용할 구분자 문자입니다. 기본값은 '_'입니다.
Returns
: 키가 기준 키와 지정된 모드에 따라 일치하면 true를 반환하고, 그렇지 않으면 false를 반환합니다.

이 함수는 주어진 키와 기준 키를 지정된 모드와 구분자를 기준으로 비교합니다.

  • 모드 0에서는 전체 키를 직접 비교합니다.
  • 모드 1에서는 구분자 앞부분의 키와 기준 키의 앞부분을 비교합니다.
  • 모드 2에서는 구분자 뒷부분의 키와 기준 키의 뒷부분을 비교합니다.
  • 구분자를 제공하지 않을 경우 자동으로 모드 0으로 동작합니다.
